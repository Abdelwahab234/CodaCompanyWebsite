# E-Book Library Management System

نظام متكامل لإدارة المكتبات الرقمية تم تطويره باستخدام Java Swing و SQL Server، ويوفر إدارة كاملة للكتب والمؤلفين ودور النشر من خلال واجهة سهلة الاستخدام وأداء موثوق.

### أبرز المميزات

• لوحة تحكم حديثة وسهلة الاستخدام.
• نظام تسجيل دخول وصلاحيات آمن.
• إدارة كاملة للكتب والمؤلفين والناشرين.
• محرك بحث متقدم متعدد المعايير.
• تقارير وإحصائيات تفصيلية.
• نظام نسخ احتياطي وحماية للبيانات.

### التقنيات المستخدمة

• Java SE 8+ (Swing)
• SQL Server 2016+
• OOP & Database Design

### القيمة المضافة

يساعد النظام المكتبات الصغيرة والمتوسطة على تنظيم المحتوى الرقمي، تسريع عمليات البحث والإدارة، وتحسين كفاءة العمل من خلال واجهة بسيطة وقابلة للتوسع.
